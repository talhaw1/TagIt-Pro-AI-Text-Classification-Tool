# TagIt-Pro-AI-Text-Classification-Tool
Python, Hugging Face Transformers, PyTorch, FastAPI, Gradio
• Built a zero-shot text classification tool using facebook/bart-large-mnli; exposed predictions via a FastAPI REST endpoint and a Gradio interactive interface.
• Implemented confidence thresholding and semantic similarity checks to flag low-confidence outputs, improving prediction
reliability.
• Enabled batch processing of large text datasets with automated CSV export and summary analytics.
• Wrote clean, well-documented code with modular pipeline design to support easy reuse and extension.
